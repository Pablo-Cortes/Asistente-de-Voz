🎙️ Aura Voice Hybrid

Aura Voice Hybrid es un asistente de voz interactivo basado en tecnologías web que permite:

Convertir texto a voz (Text-to-Speech)
Convertir voz a texto (Speech-to-Text)
Modo eco automático (repite lo que dices)

Todo esto con una interfaz moderna, minimalista y responsiva.

🚀 Características

✨ Interfaz futurista con animaciones
🎤 Reconocimiento de voz en tiempo real
🔊 Síntesis de voz configurable
🌐 Soporte para múltiples idiomas (español / inglés)
🔁 Modo eco automático
📱 Optimizado para dispositivos móviles

🧠 Tecnologías utilizadas
HTML5
CSS3 (Glassmorphism + animaciones)
JavaScript (Vanilla)
Web Speech API
SpeechRecognition
SpeechSynthesis
⚙️ Funcionamiento

El sistema combina dos APIs principales del navegador:

Speech-to-Text: Convierte tu voz en texto usando el micrófono
Text-to-Speech: Convierte texto en audio utilizando voces disponibles del sistema

Puedes ver la implementación directamente en el archivo principal del proyecto

🎮 Cómo usar
Abre el proyecto en tu navegador
Escribe texto y presiona "Leer Texto" 🔊
O toca el círculo central para comenzar a hablar 🎤
Activa el modo eco para que el sistema repita automáticamente lo que dices
📌 Requisitos
Navegador moderno (Google Chrome recomendado)
Permisos de micrófono habilitados
Conexión a internet (en algunos navegadores)
⚠️ Limitaciones
Algunas funciones dependen del soporte del navegador
SpeechRecognition no está disponible en todos los navegadores
Las voces disponibles pueden variar según el sistema operativo
